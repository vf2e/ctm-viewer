# Mag-aim 软件设计风格规范

> 适用范围：**Mag-aim 导航侧**（`main2.qml` 第二屏、`ui/Pages2/Qml/` 目录）。  
> 不包含 TMS 治疗页（`ui/Pages/Treatment`、`ui/Pages/Test` 等旧界面），那些界面不在本规范管辖范围。  
> 组件实现细则见同目录 [`公共组件编写规范.md`](./公共组件编写规范.md)。

---

## 1. 设计定位

Mag-aim 界面采用 **扁平化、浅色层级、低干扰** 的视觉语言，服务于术中导航与配置场景：

- 以白/浅灰蓝为底，蓝色为主操作色，红色仅用于危险与错误。
- 信息通过 **卡片分区 + 细边框 + 轻阴影** 组织，避免厚重渐变与强对比色块。
- 交互控件高度、圆角、边框尽量统一，减少「同一页面多种按钮样式」。
- 所有尺寸应通过 `windowRoot2.pxToActual()` 做分辨率适配，禁止写死物理像素后到处复制。

## 2. 色彩系统

### 2.1 基础色（页面级 `readonly property`，抽屉/弹窗应复用同名变量）

| 语义 | 色值 | 用途 |
|------|------|------|
| 页面背景 `clrBg` | `#F5F8FD` | 抽屉、导航设置等内容区底色 |
| 卡片背景 `clrCard` | `#FFFFFF` | 卡片、弹窗、表头容器 |
| 边框 `clrBorder` | `#E2E8F0` | 卡片描边、分割线 |
| 标题文字 `clrTitle` | `#0F172A` | 页面标题、弹窗标题、表头强调 |
| 正文文字 `clrText` | `#334155` | 表单、列表正文、次按钮文字 |
| 辅助文字 `clrSubText` | `#64748B` | 标签、说明、滑条标题 |
| 弱化文字 | `#94A3B8` | 占位符、禁用态、图标次要态 |

登录页背景可使用 `#F8FAFC`，仍属同一色系，不单独引入新主色。

### 2.2 主色（操作与选中）

| 语义 | 色值 | 用途 |
|------|------|------|
| 主色 `clrPrimary` | `#3B82F6` | 主按钮、选中描边、滑条填充 |
| 主色悬停 | `#2563EB` | 主按钮 hover、Header 操作钮 |
| 主色按下 | `#1D4ED8` | 主按钮 pressed |
| 主色浅底 `primarySoft` | `#DBEAFE` | **表格/列表选中行**（推荐） |
| 主色更浅底 | `#EFF6FF` | Tab 选中、下拉选中项、信息提示区 |
| 焦点描边 | `#93C5FD` | 输入框 focus（非错误时） |

### 2.3 表单控件

| 语义 | 色值 |
|------|------|
| 输入框背景 `clrFieldBg` | `#F8FAFC` |
| 输入框边框 `clrFieldBorder` | `#CBD5E1` |

### 2.4 语义色（结果 / 危险 / 警告）

| 类型 | 图标底 | 图标/文字色 | 场景 |
|------|--------|-------------|------|
| 成功 | `#DCFCE7` | `#16A34A` | `showSuccess`、成功图标 |
| 警告 | `#FEF3C7` | `#D97706` | 确认框 `iconTone: "warn"` |
| 错误/危险 | `#FEE2E2` | `#DC2626` | 删除确认、表单错误 |
| 信息 | `#EFF6FF` | `#2563EB` | 一般提示 |

危险按钮（`FlatButton` `tone: "danger"`）：

- 文字 `#EF4444`，边框 `#FCA5A5`，悬停底 `#FFF1F2`。

### 2.5 遮罩

- 模态遮罩统一：`#5A0F172A`（约 35% 透明黑）
- 卡片阴影：`#12000000` ~ `#16000000`，`verticalOffset: 2~3`

## 3. 字体与字号

系统默认使用 Qt 默认字体族，通过 `font.pixelSize` 控制层级（均需 `pxToActual`）：

| 层级 | 推荐字号 | 字重 | 示例 |
|------|----------|------|------|
| 抽屉主标题 | 22 | bold | 「用户中心」「设置中心」 |
| 弹窗/区块标题 | 18 | bold | 确认框标题、消息提示标题 |
| 正文 / 按钮 / 输入 | 14 | regular / bold* | 表单、FlatButton 默认 |
| 辅助标签 | 12~13 | regular | Header 字段标签、表格小按钮 |
| 弱化说明 | 12 | regular | 登录页副标题 |

\* `FlatButton`：`primary` / `danger` 默认 **bold**，`secondary` 为 regular。

- 正文行高：弹窗说明文字 `lineHeight: 1.35`。
- 长文本：`wrapMode: Text.WordWrap`，标题 `maximumLineCount: 2` 时配合 `elide` 策略。

## 4. 间距、尺寸与圆角

### 4.1 标准控件高度

| 控件 | 高度 |
|------|------|
| 按钮 / 输入框 / 下拉框 / 复选框行 | **38px**（`pxToActual(38, "y")`） |
| Tab 按钮 | 36 |
| 抽屉内紧凑操作钮 | 32~34（仅表格行内等特殊场景） |

### 4.2 常用间距

| 场景 | 水平 | 垂直 |
|------|------|------|
| 抽屉外边距 | 16 | 16 |
| 抽屉内区块间距 | — | 12 |
| 卡片内边距 | 14~16 | 14~16 |
| 表单弹窗边距 | 24~28 | 20 |
| 消息/确认弹窗 | **24** | **20** |
| 图标与标题间距 | 12 | — |
| 标题与正文间距 | — | 8 |
| 按钮组间距 | 12 | — |

### 4.3 圆角

| 元素 | 圆角 |
|------|------|
| 卡片 / 弹窗 / 抽屉标题条 | **12** |
| 按钮 / 输入框 / Tab | **8** |
| 复选框指示器 | 6 |
| 关闭图标按钮 | 10~12 |
| 登录大面板 | 20 |

### 4.4 边框

- 默认：`border.width: 1`，`border.color: #E2E8F0` 或 `#CBD5E1`（表单）。
- 主按钮：**无边框**；次按钮 / 危险按钮：1px 描边。

## 5. 阴影规范

仅在需要「浮起」的容器使用阴影，普通嵌套卡片可关闭：

```qml
layer.enabled: true
layer.effect: DropShadow {
    transparentBorder: true
    radius: 14
    samples: 18~20
    color: "#12000000"   // 或 "#16000000"
    verticalOffset: 3
}
```

| 场景 | radius | verticalOffset |
|------|--------|----------------|
| 抽屉标题条 / 业务弹窗 | 14 | 3 |
| `FlatCard`（`useShadow: true`） | 10 | 2 |
| 登录面板 | 32 | 10 |

## 6. 布局模式

### 6.1 右侧抽屉（用户 / 历史 / 方案 / 设置）

```
┌─────────────────────────────────────────┐
│  [标题卡片 56h]              [关闭图标] │
├─────────────────────────────────────────┤
│  [筛选条 / Tab 卡片]                    │
├─────────────────────────────────────────┤
│  [主内容卡片 / 表格 / 分栏]              │
└─────────────────────────────────────────┘
```

- 背景：`clrBg`（`#F5F8FD`）。
- 外边距 16，区块间距 12。
- 标题区：白底圆角 12 卡片 + 轻阴影；右侧 **`DrawerCloseButton`**（抽屉级关闭，与弹窗内关闭区分）。
- 宽度参考：用户抽屉 `1080`（`pxToActual`）。

### 6.2 顶栏 Header（三维区浮动）

- 白底半透明（`opacity: 0.95`）+ 圆角 12 + 轻阴影。
- 高度约 64；内边距左右 20；字段区间距 24。
- 与患者相关的激活态浅蓝底：`#EFF6FF`。

### 6.3 居中业务弹窗（编辑用户、方案管理等）

- 挂 **`Overlay.overlay`**，使用 `centerOnWindow()` 相对整窗居中（不要放在 Drawer 内部再 `Overlay.overlay`，避免定位异常）。
- 典型宽度：表单类 780、选择用户 860 等，按内容定宽。
- 标题行：左标题 + 右 `DrawerCloseButton`（业务弹窗允许右上角关闭；**消息/确认类弹窗不要**）。
- 底栏：右对齐「取消 + 确定」或「保存」。

### 6.4 消息提示与二次确认（统一样式）

`FlatMessagePopup` 与 `FlatConfirmDialog` **必须保持同一视觉规格**：

| 属性 | 值 |
|------|-----|
| 宽度 | 400 |
| 边距 | 水平 24 / 垂直 20 |
| 图标容器 | 44×44 圆形 |
| 图标图片 | 24×24（`success.png` / `error.png` / `warn.png`） |
| 标题字号 | 18 bold |
| 正文字号 | 14，颜色 `#475569`，相对标题左侧缩进 `iconBox + 12` |
| 分割线 | 1px `#E2E8F0` |
| 按钮区高度 | `38 + 20×2` |
| 按钮 | 高 38；主按钮宽 104，次按钮宽 92 |

- **无右上角关闭**；关闭方式：底部按钮、Esc、点击遮罩。
- 消息提示：仅「知道了」；确认框：「取消」+「确认」。

全局单例：`main2.qml` → `windowRoot2.messagePopup`。

### 6.5 气泡锚点弹窗（如导航设置）

- 使用 `CustomPopup`，白底圆角 12，边框 `#E2E8F0`。
- 相对触发控件定位，左侧三角尖头对准触发图标中心（`arrowOffset` 动态计算）。

## 7. 组件库（必须使用）

新页面 **禁止** 直接使用原生 `Button` / `TextField` / `TabButton`（抽屉与 Pages2 新弹窗范围内）。

| 组件 | 用途 |
|------|------|
| `FlatButton` | 所有按钮 |
| `FlatTextField` | 单行输入 |
| `FlatDatePicker` | 日期选择 |
| `FlatComboBox` | 下拉选择 |
| `FlatTabButton` | TabBar 标签 |
| `FlatCheckBox` | 复选框 |
| `FlatCard` | 可复用卡片容器 |
| `FlatSliderRow` | 标签 + 滑条 + 数值 + 加减 |
| `FlatProgressBar` | 水平进度条 |
| `FlatConfirmDialog` | 二次确认 |
| `FlatMessagePopup` | 操作结果提示（挂 main2） |
| `DrawerCloseButton` | 抽屉/业务弹窗右上角关闭 |

### 7.1 按钮语义 `tone`

| tone | 用途 | 示例 |
|------|------|------|
| `primary` | 主操作 | 查询、保存、确认、知道了 |
| `secondary` | 次操作 | 取消、重置、刷新、翻页 |
| `danger` | 危险操作 | 删除、停止 |

### 7.2 输入与下拉

- 背景 `#F8FAFC`，圆角 8，高度 38。
- 默认边框 `#CBD5E1`；聚焦 `#93C5FD`（TextField）或主色（ComboBox）。
- 错误：底 `#FEF2F2`，边框 `#EF4444` / `#FCA5A5`。

### 7.3 Tab 选中态

- 选中：底 `#DBEAFE`，边框 `#93C5FD`，文字 `#1D4ED8` bold。
- 未选中：透明底，文字 `#334155`。
- 必须：`selected: tabBar.currentIndex === index`。

## 8. 表格与列表

### 8.1 推荐选中样式（新代码统一）

- 选中行背景：**`#DBEAFE`**，文字保持 **`#334155`**（不用深蓝底白字）。
- 悬停行：`#F1F5F9` 或 `#F8FAFC`。
- 表头背景：`#EFF6FF` 或 `#F7F8FA`，表头文字 `#64748B`。

### 8.2 分割线

- 行分割：`#F1F5F9`。
- 区块分割：1px `#E2E8F0` 通栏线（与弹窗底部分割一致）。

## 9. 图标与反馈

### 9.1 图标资源

- 关闭：`qrc:/ui/Pages2/Assets/Images/close.png`
- 结果图标：`success.png` / `error.png` / `warn.png`
- 禁用 Emoji 做状态图标（布局不稳定）；统一 PNG 或几何字符（`!` `✓` `✕`）放在固定圆形容器内。

### 9.2 反馈层级

| 层级 | 方式 | 场景 |
|------|------|------|
| 轻提示 | `CustomMessage`（toast，顶部居中） | 非阻塞、单行短时通知 |
| 通知 | `notification`（`FlatNotification`，默认右上角，可改右下角） | 标题+描述、可关闭；`placement` / `setPlacement()` |
| 结果弹窗 | `messagePopup.showSuccess/Error/...` | 操作完成需用户知晓 |
| 二次确认 | `FlatConfirmDialog` | 删除、结束疗程等 |
| 表单错误 | 字段旁/表单底错误文案 + 输入框 `hasError` | 校验失败 |

## 10. 交互与动效

- 可点击区域：`cursorShape: Qt.PointingHandCursor`。
- 禁用：`opacity: 0.55` 或按钮内置灰色态 `#94A3B8`。
- `DrawerCloseButton` / 登录关闭钮：悬停浅灰底 `#F1F5F9`，按下浅红底 `#FEE2E2`，轻微 `scale`（0.94~1.02），过渡 120~140ms。
- 避免大面积闪烁动画；表单错误可用短时位移动画（shake）强调。

## 11. 代码组织约定

```
ui/Pages2/Qml/
├── common/          # Flat* 组件、规范文档
├── components/      # 抽屉、业务弹窗、功能页
├── ModelView.qml    # 三维主视图
├── Header.qml       # 顶栏
└── main2.qml        # 窗口根、全局弹窗单例
```

- 新组件命名：`FlatXxx.qml`，加入 `res.qrc`，**qmake + 全量重编**（启用 QML 预编译时必需）。
- 第三次以上重复样式 → 提取到 `common/`。
- 页面内定义颜色时，优先复用本节 **第 2 节** 命名（`clrCard`、`clrPrimary` 等），与 `UserModuleDrawer` 保持一致。

## 12. 禁止事项

1. 在 Pages2 新抽屉页使用原生 `Button` / `TextField` / `TabButton` 并手写一套新样式。
2. 在业务页内新建 `Popup` 做成功/失败提示（应使用 `windowRoot2.messagePopup`）。
3. 在 `FlatMessagePopup` / `FlatConfirmDialog` 加右上角关闭按钮。
4. 在 Drawer 子树里对 `Overlay.overlay` 弹窗做「伪居中」而不挂到 `main2` 窗口层。
5. 引入 TMS 旧页面的渐变按钮、深色大色块、未适配的固定像素布局。
6. 选中表格行使用高饱和深蓝底 `#3498db` + 白字（Legacy 默认，新界面勿再沿用）。
